# BazaarBuddy
A Project made for Hack Club's [Flavortown](https://flavortown.hackclub.com/). Check them out!

BazaarBuddy is a tool created to solve Skyblock's biggest woe: the insanely convoluted crafting recipes, of course! BazaarBuddy makes a rich visualization of whatever crafting recipe you need to make, and tells you exactly what materials are needed to make it. 
## Capabilities
* Display of a tree of items, allowing you to see each stage of a craft
* Minion time estimator, allowing you to see how you can get what you need and how long it will take.
